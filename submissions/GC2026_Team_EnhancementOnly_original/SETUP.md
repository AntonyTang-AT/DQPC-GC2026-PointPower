# Environment Setup — GC2026 Enhancement Only (frame_gate v2)

## Prerequisites

- **Dataset**: UVG-CWI-DQPC (CG + HE PLY files for all 12 sequences)
- **CG_LIST**: a text file listing all CG PLY paths (one per line)

## Step-by-step

### Step 1: Set workspace root

```bash
export GC2026_ROOT=/path/to/your/workspace
```

The workspace must contain the dataset at:
```
$GC2026_ROOT/
  data/raw/UVG-CWI-DQPC/
    <Sequence>/
      consumer-grade_capture_system/CG/15fps/    ← CG PLY files
      high-end_capture_system/HE/15fps/           ← HE PLY files (for validation)
```

### Step 2: Generate pair lists

```bash
cd submissions/GC2026_Team_EnhancementOnly
bash data/generate_pair_lists.sh
```

This creates:
- `$GC2026_ROOT/data/processed/all_cg_only_cgv2.txt` — all 2155 CG frames
- `$GC2026_ROOT/data/processed/val_cg_only_official_cgv2.txt` — validation frames
- `$GC2026_ROOT/data/processed/val_pairs_official_cgv2.txt` — validation CG/HE pairs

### Step 3: Set up Python environment

```bash
conda create -n superpc python=3.9 -y && conda activate superpc
pip install -r requirements.txt
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu128
bash src/setup_pdlts_deps.sh
```

### Step 4: Provide CG list

```bash
export CG_LIST=$GC2026_ROOT/data/processed/all_cg_only_cgv2.txt
```

### Step 5: Run inference

```bash
bash src/run.sh
```

Output: `$GC2026_ROOT/output/submission_candidate_frame_gate_v2/` (2155 ENH PLY files).

### Step 6: Smoke test (optional, 2 frames)

```bash
bash src/run_smoke.sh
```

### Step 7: Re-run PD-LTS UVG fine-tune (optional)

```bash
bash src/setup_pdlts_train.sh
GPUS=4 bash src/run_pdlts_finetune_uvg.sh train
```

## Required inputs (summary)

| Variable | Purpose | Example |
|----------|---------|---------|

```
$GC2026_ROOT/
  data/raw/UVG-CWI-DQPC/             ← dataset (provided by organizer)
    <Sequence>/consumer-grade.../CG/15fps/*.ply
    <Sequence>/high-end.../HE/15fps/*.ply
  data/processed/                     ← generated by generate_pair_lists.sh
    all_cg_only_cgv2.txt
    val_cg_only_official_cgv2.txt
    val_pairs_official_cgv2.txt

```
